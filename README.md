### hexo基本命令

描述 | 命令 | 缩写
--- | --- | --- 
生成静态页面 | hexo generate | hexo g
开启服务 | hexo server | hexo s
部署到github | hexo deploy | hexo d
新建文章 | hexo new "articleName" | -
新建页面 | hexo new page "htmlName" | -
查看帮助 | hexo help | - 
查看hexo版本号 | hexo version

### 组合命令
描述 | 命令 |
--- | --- |  
生成页面并预览 | hexo s -g | 
生成页面并上传到github | hexo d -g |
